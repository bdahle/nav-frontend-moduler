# Hvordan ta i bruk ProductHeader

Denne header-komponenten er utviklet i sammenheng med nye produktsider på NAV.

```jsx
<ProductHeader>
  <ProductHeader.Wrapper>
    <ProductHeader.Heading>Arbeidsavklaringspenger (AAP)</ProductHeader.Heading>
    <ProductHeader.Description>
      LABORIS AUTE DESERUNT ID Arbeidsavklaringspenger (AAP) EX ET CUPIDATAT
      CONSEQUAT DOLORE.
    </ProductHeader.Description>
  </ProductHeader.Wrapper>
</ProductHeader>
```

## Med illustrasjon

Illustrasjonene brukt i header er "Pictogram" som er designet for den sepsifikke siden og
man skal da ikke bare velge et fra figma vilkårlig.

<Canvas>
